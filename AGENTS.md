# AGENTS.md — Реєстр агентів і порядок запуску

> Читай цей файл перед запуском будь-якого агента.
> Агенти запускаються ПОСЛІДОВНО — не паралельно.

---

## Як запускати агентів

Кожен агент запускається з кореня проєкту командою:

```bash
claude --print agents/01_project_setup.md
```

Або в інтерактивному режимі Claude Code:
1. Відкрий термінал у папці `audio-quiz-platform/`
2. Запусти `claude`
3. Напиши: `Read agents/01_project_setup.md and execute all tasks`

**Після кожного агента:** перевір що файли створені, запусти тести якщо є, тільки тоді переходь до наступного.

## Якщо агент застряг або зробив помилку

1. Не запускай наступного агента
2. Опиши помилку в чаті Claude Code
3. Попроси виправити конкретний файл
4. Перевір ще раз перед продовженням

---

## Перевірка після всіх агентів

```bash
# 1. Встановити залежності
pip install -r requirements.txt

# 2. Застосувати міграції
alembic upgrade head

# 3. Запустити сервер
uvicorn app.main:app --reload

# 4. Відкрити браузер
open http://localhost:8000
```
